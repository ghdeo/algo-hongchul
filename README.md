# Rules
1. cpp 파일 형식은 `{문제번호}_{이름}.cpp` 형식으로 작성
2. 코드 작성시 주석 추가 요망
3. 커밋 메시지는 `[{이름}] {문제번호}번 {메시지}` 형태로 작성 (ex. \[홍길동] 105번 풀이)
* Deadline: Thursday 19:00
* Directory name: /week0/your_name
